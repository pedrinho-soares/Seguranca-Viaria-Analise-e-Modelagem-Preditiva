# Seguranca Viaria Analise e Modelagem Preditiva

### Segurança Viária — Setembro: Análise e Modelagem Preditiva

Este projeto apresenta uma análise dos acidentes de trânsito ocorridos em setembro, com foco em **segurança viária** e **modelagem preditiva**. O objetivo é identificar padrões temporais e espaciais, traçar o perfil dos veículos envolvidos, avaliar a gravidade dos acidentes e testar modelos de classificação para prever a gravidade.

---

##  Objetivos
- Identificar os trechos mais críticos em rodovias.
- Analisar o perfil dos veículos envolvidos.
- Avaliar a gravidade dos acidentes (mortos, feridos, ilesos).
- Ajustar modelos de classificação para prever gravidade.
- Comparar resultados com métricas de avaliação (acurácia, precisão e recall).
- Propor recomendações práticas para campanhas educativas e fiscalização.

---

##  Tecnologias utilizadas
- Python (pandas, seaborn, matplotlib, scikit-learn)
- Jupyter Notebook

---

##  Principais resultados
- Acidentes concentrados em dias úteis e horários de pico.
- Motocicletas e caminhões associados a maior gravidade.
- Modelos preditivos testados (Logistic Regression, Random Forest, KNN, SVM) com acurácia em torno de 60%.
- Limitações dos dados atuais reforçam a necessidade de variáveis mais completas para análises robustas.

---

##  Considerações finais
Os dados de setembro revelam padrões importantes, mas também destacam os limites da modelagem preditiva com as variáveis disponíveis.  
A combinação de **ações imediatas** (fiscalização e campanhas) com **investimento em infraestrutura e coleta de dados mais ricos** é essencial para reduzir acidentes e salvar vidas.
